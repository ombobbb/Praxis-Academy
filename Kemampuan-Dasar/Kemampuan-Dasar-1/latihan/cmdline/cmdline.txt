ipconfig = Untuk menampilkan konfigurasi dasar TCP / IP untuk semua adaptor 
Dir = Untuk menampilkan daftar folder yang berada pada suatu komputer
Dir (Nama Folder) = Untuk menampilkan isi folder yang dipanggil
D: = untuk berpindah dari Localdisk C ke Localdisk D
cd (nama directory) = untuk  masuk ke dalam directory yang dipanggil
cd .. = untuk kembali ke folder induk sebelumnya
move = untuk memindahkan file 
copy = untuk mengcopy suatu file dan folder ke lokasi tertentu yang diinginkan.
ping (alamat url) = untuk mengecek konektivitas internet
date = untuk mengecek tanggal
mkdir = untuk membuat folder baru
date = Perintah yang digunakan untuk mengatur kalender pada komputer
dxdiag = untuk mengetahui atau mengecek spesifikasi pada komputer. Mulai dari kapasitas RAM, Jenis Processor, Jenis Bios dan Sistem Operasi Yang Digunakan.
notepad = untuk membuka notepad
taskmgr = untuk membuka task manager
color = mengganti warna tampilan CDM
calc = membuka kalkulator
tasklist = untuk menampilkan daftar program yang aktif
taskkill = untuk menutup paksa program yang aktif
CLS = perintah untuk membersihkan layar Command Prompt yang sudah berisi perintah-perintah yang digunakan.
